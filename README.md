# Testv6

Unity 6000.2 project (XR/Meta Quest).

## Requirements

- **Unity**: 6000.2.14f1 or compatible
- **Platform**: Android (Meta Quest)

## Setup

1. Clone the repo:
   ```bash
   git clone <your-repo-url>
   cd Testv6
   ```
2. Open the project in Unity Hub or Unity Editor.
3. Let Unity import assets and regenerate the `Library` folder.

## Git / GitHub

- **`.gitignore`**: Unity-specific (Library, Temp, Logs, builds, IDE files, etc.).
- **`.gitattributes`**: Text/binary rules for assets and line endings.

To initialize Git locally (if not already):

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```
